## Overview

This project is a full-stack web application designed to showcase my skills in web development and provide a practical demonstration of a Shoes selling site. It allows users to interact with the application by creating accounts, adding shoes to cart, adding shos to wishlist, and a search system for shoes.


## Features

- <h2>User Registration and Authentication:</h2> Users can create accounts with secure authentication. This feature ensures data privacy and personalized experiences.

- <h2>Seamless Shopping Experience::</h2> Explore and select your preferred shoes from a wide array of styles and brands.Add items to your cart, see real-time price calculations, and experience the shopping process from start to finish, even if it's for demonstration purposes.

- <h2>Wishlist Functionality:</h2> Save your favorite shoes to your wishlist for future reference.

- <h2>Responsive Design:</h2> The application is built with a responsive design to ensure optimal user experience across various devices, including desktops, tablets, and mobile phones.KickKart's responsive design ensures optimal user experiences across various devices.

- <h2>Database Integration:</h2> The backend of the application connects to a database to store user profiles, shoes. 

- <h2>Server-Side Logic:</h2> The backend of the application contains server-side logic for handling user authentication, sending shoes data to client.

- <h2>Sending Otp Emails:</h2> The app is also capable to send email to users which consist of an OTP code to authenticate users.
